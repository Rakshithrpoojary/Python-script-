# AWS Python Automation Scripts

This repository contains automation scripts written in Python using boto3.

## Features
- Stop/Start EC2 instances
- Create/Delete S3 buckets
- Cleanup unused EBS volumes
- IAM user creation
- Read CloudWatch logs
- EC2 backup automation

## Requirements
- Python 3.x
- boto3
- AWS credentials configured

## Setup
pip install boto3
aws configure
